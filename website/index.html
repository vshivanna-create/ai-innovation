How It Works

1

Push Code

Developer pushes code to main branch, triggering GitHub Actions workflow automatically.

2

Security Scan

Three tools run in parallel: Gitleaks for secrets, Semgrep for code issues, OPA for policies.

3

AI Analysis

AI aggregates all findings, understands context, and makes an intelligent deploy/block decision.

4

Deploy or Block

If safe, deploys to S3 + CloudFront. If risky, blocks and provides detailed remediation steps.
